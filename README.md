
# 🗑️Iot-BASED SMART PLASTIC WASTE MONITORING SYSTEM 

## 📌 Overview

This project presents a **Smart IoT-Based Dustbin System** designed to improve plastic waste management in colleges and hostels. The system automates dustbin operation, monitors waste levels in real time, and provides alerts to maintain cleanliness and hygiene.



## 🎯 Problem Statement

Traditional dustbins do not have monitoring capabilities, which leads to overflow, unhygienic conditions, and delayed waste collection. There is no proper system to track waste levels or create awareness about plastic disposal.



## 💡 Proposed Solution

We developed a smart dustbin system that:

* Automatically opens when a user approaches (touch-free)
* Measures waste using sensors
* Displays real-time data
* Detects when the bin is full
* Sends alerts using IoT



## ⚙️ Features

* Automatic lid opening using ultrasonic sensor
* Waste level detection
* Real-time monitoring
* Mobile notification using IoT
* Voice alert system
* User-friendly and low-cost design



## 🏗️ System Architecture

```
User → Sensor → NodeMCU → 
→ Servo Motor (Lid Control)
→ Load Cell (Weight Measurement)
→ LCD Display
→ Buzzer / Mobile Notification
```



## 🔧 Hardware Requirements

* NodeMCU (ESP8266)
* Ultrasonic Sensors (2)
* Load Cell + HX711
* Servo Motor
* LCD Display (16x2 I2C)
* Buzzer / Speaker
* Power Supply
* Breadboard & Jumper Wires



## 💻 Software Requirements

* Arduino IDE
* Embedded C
* ESP8266 Libraries
* Blynk IoT App



## 🔄 Working Principle

* The ultrasonic sensor detects the user and opens the lid automatically
* Waste is measured using a load cell
* Data is displayed on the LCD
* When the bin is full, an alert is triggered
* Notification is sent to mobile using IoT



## 🧪 Implementation & Prototype

The system is implemented using NodeMCU connected with sensors and actuators. The prototype is built on a breadboard and tested for real-time operation.



## 💻 Code Explanation

* WiFi connection setup using ESP8266
* Sensor data collection (ultrasonic + load cell)
* Servo motor control for lid
* Condition check for full bin
* Alert system using buzzer and Blynk notification
